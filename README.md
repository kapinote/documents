# KapiNote Documents

[https://docs.kapinote.co](https://docs.kapinote.co)

## Getting started

Install NVM

```
brew install nvm
```

Install Node v18+

```
nvm install v18
```

Install Bun

```
npm install -g bun
```

Install packages

```
bun install
```

Start dev

```
bun run docs:dev
```
